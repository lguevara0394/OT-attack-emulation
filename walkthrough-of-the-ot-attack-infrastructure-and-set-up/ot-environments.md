# OT Environments

**The OT environment:**\
For the OT environment we first used Labshock for the first experiment using this github\
link for the installation:\
https://github.com/zakharb/labshock/wiki/Quickstart-Guide\
However, it is important to notice that once registered the labshock environment is free only for\
30 days trial period.\
For this reason, we used the Vicsort testbed for the second half of the experiment in order to\
have an open sourced OT environment. Please use the steps indicated in the following gitlab\
link:\
https://gitlab.com/ekisac10/vicsort/-/wikis/VICSORT-Setup-Guide
